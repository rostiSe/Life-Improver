# T-012 — AI weekly agent

**Status**: backlog
**Type**: feat

---

## Goal

Build the weekly AI agent that runs Sunday at 23:00 UTC via Vercel Cron. It generates a structured weekly review covering accomplishments, patterns, and next week's priorities.

---

## Acceptance criteria

- [ ] Vercel Cron job configured in `vercel.json` to run `POST /api/cron/weekly` Sunday at 23:00 UTC
- [ ] Route protected by `Authorization: Bearer ${CRON_SECRET}` check
- [ ] Agent collects: all journal entries, completed tasks, habit logs, calendar events for the past 7 days
- [ ] Agent also queries the embedding table for semantically related content from past weeks
- [ ] Agent calls `streamText` with a structured weekly review prompt (accomplishments, blockers, patterns, next week priorities)
- [ ] Review written to `agent_log` (type: "weekly") in Markdown format
- [ ] Dashboard `/` surfaces the latest weekly review in the "Latest agent insight" section
- [ ] Unit test for the data collection step (mock DB)
- [ ] All tests green
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
