# T-004 — Task module

**Status**: backlog
**Type**: feat

---

## Goal

Build the task module: create, view, update, and complete tasks. Tasks can be linked to a project.

---

## Acceptance criteria

- [ ] `/tasks` route lists all tasks grouped by status (todo, in_progress, done)
- [ ] Inline task creation from the list view
- [ ] Task detail drawer/modal for editing title, description, due date, status, project link
- [ ] Task status updated via Server Action
- [ ] Completed tasks show a completion timestamp
- [ ] Tasks can be filtered by status
- [ ] `entity` row created on task save (type: "task")
- [ ] Task title embedded on save
- [ ] Zod schema in `src/modules/Task/schemas/` is single source of truth for form types
- [ ] All components follow file structure rules
- [ ] All tests green
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
