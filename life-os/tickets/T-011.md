# T-011 — AI daily agent

**Status**: backlog
**Type**: feat

---

## Goal

Build the daily AI agent that runs at midnight via Vercel Cron. It retrieves today's data, calls OpenAI via Vercel AI SDK, and writes a summary to `agent_log`.

---

## Acceptance criteria

- [ ] Vercel Cron job configured in `vercel.json` to run `POST /api/cron/daily` at midnight UTC
- [ ] Route protected by `Authorization: Bearer ${CRON_SECRET}` check
- [ ] Agent collects: today's journal entries, completed tasks, habit logs, calendar events
- [ ] Agent calls `streamText` with a structured system prompt
- [ ] Agent writes the result to `agent_log` (type: "daily", status: "completed" or "failed")
- [ ] `agent_log` row includes `inputTokens`, `outputTokens`, `durationMs`
- [ ] On error: `agent_log` row written with status "failed" and error message
- [ ] Unit test for the data collection step (mock DB)
- [ ] All tests green
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
