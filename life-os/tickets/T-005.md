# T-005 — Project module

**Status**: backlog
**Type**: feat

---

## Goal

Build the project module: create and manage projects, view linked tasks, and update project status.

---

## Acceptance criteria

- [ ] `/projects` route lists all projects grouped by status (active, paused, completed, archived)
- [ ] `/projects/[id]` shows project detail with name, description, status, due date
- [ ] Project detail lists all linked tasks with inline status update
- [ ] New project form with name (required), description (optional), due date (optional)
- [ ] Project status updated via Server Action
- [ ] `entity` row created on project save (type: "project")
- [ ] Project name embedded on save
- [ ] Zod schema in `src/modules/Project/schemas/` is single source of truth for form types
- [ ] All components follow file structure rules
- [ ] All tests green
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
