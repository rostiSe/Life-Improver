# T-009 — Dashboard

**Status**: backlog
**Type**: feat

---

## Goal

Build the dashboard — the home screen of LifeOS. Shows today's tasks, today's habits, upcoming events, and the most recent journal entry.

---

## Acceptance criteria

- [ ] `/` route is the dashboard
- [ ] "Today's tasks" section: lists tasks due today or overdue, with inline status toggle
- [ ] "Today's habits" section: lists daily habits with today's completion count and Log button
- [ ] "Upcoming events" section: next 5 events from today
- [ ] "Recent journal" section: snippet of the most recent journal entry with link to full entry
- [ ] "Latest agent insight" section: most recent `agent_log` summary (if any)
- [ ] All data fetched in a Server Component; no client-side fetching
- [ ] Page renders correctly when all sections are empty
- [ ] All components follow file structure rules
- [ ] All tests green
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
