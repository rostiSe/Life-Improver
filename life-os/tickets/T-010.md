# T-010 — Semantic search & embeddings

**Status**: backlog
**Type**: feat

---

## Goal

Enable semantic search across all content. Every entity's text content is embedded on save using `text-embedding-3-small`. A search bar queries the embedding table using pgvector cosine similarity.

---

## Acceptance criteria

- [ ] Embedding utility in `src/lib/embeddings.ts` — wraps `@ai-sdk/openai` embed call
- [ ] Embedding is triggered as a non-blocking background task after any entity save
- [ ] `embedding` table populated with correct `entity_id`, `model`, `content`, `embedding` (vector)
- [ ] Search route `/api/search` accepts `query` param, returns top 10 similar results
- [ ] pgvector `<=>` cosine distance operator used in Drizzle raw query
- [ ] Search results include entity type, id, and a content snippet
- [ ] Global search input in the navigation shell (from T-002)
- [ ] Search results displayed in a command-palette style overlay
- [ ] All tests green (unit test the embedding utility with a mock)
- [ ] `npm run build` clean
- [ ] `npm run lint` clean
